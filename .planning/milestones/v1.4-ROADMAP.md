# Milestone v1.4: Odds Comparison UX

**Status:** SHIPPED 2026-01-26
**Phases:** 28-30
**Total Plans:** 3

## Overview

Complete redesign of the Matches page table structure with bookmaker rows, Double Chance market, and per-market margins for better odds comparison UX.

## Phases

### Phase 28: Table Restructure

**Goal**: Transform table from bookmakers-as-columns to bookmakers-as-rows (3 rows per match with rowspan layout), reorder columns, update match name format
**Depends on**: Previous milestone complete
**Plans**: 1 plan

Plans:
- [x] 28-01: Restructure table to bookmakers-as-rows layout

**Details:**
- Restructured table from horizontal bookmaker columns to vertical bookmaker rows (3 rows per match)
- Implemented rowspan cells for Region, Tournament, Kickoff, Match columns spanning all bookmaker rows
- Changed match name format from two-line "Home Team / vs Away Team" to single-line "Team A - Team B"
- Preserved all markets (1X2, O/U 2.5, BTTS) with market group headers and outcome sub-headers
- Duration: 8 min

### Phase 29: Double Chance & Margins

**Goal**: Add Double Chance market columns (1X, X2, 12) and per-market margin display per bookmaker row
**Depends on**: Phase 28
**Plans**: 1 plan

Plans:
- [x] 29-01: Double Chance columns and per-market margin display

**Details:**
- Added Double Chance (4693) to inline markets in backend
- Added margin field to InlineOdds schema with calculation
- Double Chance selectable in column settings (opt-in, not default)
- Per-market margin column displayed after each market's outcomes
- Comparative margin color coding (green = Betpawa lower, red = Betpawa higher)
- Clear border dividers between market column groups
- Duration: 12 min

### Phase 30: Page Rename & Polish

**Goal**: Rename "Matches" to "Odds Comparison" in navigation and page title, final polish
**Depends on**: Phase 29
**Plans**: 1 plan

Plans:
- [x] 30-01: Page title, back link, icon, and URL route updates

**Details:**
- Renamed page title from "Matches" to "Odds Comparison"
- Updated back link text to "Back to Odds Comparison"
- Changed navigation icon from List to BarChart3
- Changed URL route from /matches to /odds-comparison (user-approved)
- Duration: 4 min

## Milestone Summary

**Key Decisions:**
- Bookmakers-as-rows layout with rowspan for match grouping
- Margin color coding uses comparative logic (vs competitors) not absolute thresholds
- Text color for margins instead of background for readability
- BarChart3 icon chosen for better visual representation of odds comparison
- URL changed to /odds-comparison for full naming consistency
