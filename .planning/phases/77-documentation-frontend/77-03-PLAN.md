# Betpawa Odds Comparison Tool

Brief description of what the tool does and its value proposition.

## Features

- Real-time odds scraping from BetPawa, SportyBet, Bet9ja
- Cross-platform event matching using SportRadar IDs
- Side-by-side odds and margin comparison
- Historical odds tracking with visualization
- WebSocket-powered live updates
- Configurable scraping schedule and data retention

## Architecture

### Backend (FastAPI)

Brief overview of backend architecture:
- API layer (`/src/api`) - FastAPI routes, schemas, WebSocket
- Data layer (`/src/db`) - SQLAlchemy models, PostgreSQL
- Scraping layer (`/src/scraping`) - Event-centric parallel scraping
- Caching layer (`/src/caching`) - In-memory odds cache
- Scheduling layer (`/src/scheduling`) - APScheduler background jobs

### Frontend (React)

Brief overview of frontend architecture:
- Feature-based organization (`/web/src/features`)
- TanStack Query for data fetching
- WebSocket hooks for real-time updates
- shadcn/ui component library

## Quick Start

### Prerequisites

- Python 3.11+
- Node.js 18+ / pnpm
- PostgreSQL 14+

### Backend Setup

```bash
# Create virtual environment
python -m venv venv
source venv/bin/activate  # or `venv\Scripts\activate` on Windows

# Install dependencies
pip install -e .

# Set up database
createdb betpawa_comparison
alembic upgrade head

# Start server
uvicorn src.api.app:create_app --factory --reload
```

### Frontend Setup

```bash
cd web
pnpm install
pnpm dev
```

## Development

### Running Tests

```bash
# Backend
pytest

# Frontend
cd web && pnpm test
```

### Database Migrations

```bash
# Create migration
alembic revision --autogenerate -m "description"

# Apply migrations
alembic upgrade head
```
